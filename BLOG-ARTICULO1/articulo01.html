Cambiar dirección MAC de la tarjeta de red

Cambiar la dirección MAC en auditorías WiFi es importante por las siguientes razones:
         ·Anonimato: Ayuda a ocultar la identidad del auditor.
         ·Evitar Restricciones: Permite eludir bloqueos o filtros en la red objetivo.
         ·Simular Dispositivos Legítimos: Facilita la evaluación desde la perspectiva de un usuario autorizado.
         ·Evitar Registro de Actividad: Evita que tus acciones se asocien con tu dirección MAC real

La herramienta que vamos usar es  macchanger .

Para ver la MAC de una interface de red ejecutamos el comando  macchanger -s < nombre de la tarjeta de red > 

Una vez visto esto podemos usar el parametro -l para ver las direcciones MAC asociadas a las diferentes organizaciones. Grepeamos para buscar la dirección MAC de "NATIONAL SECURITY AGENCY"
